# Students CRUD Application

A simple React + TypeScript application that allows users to manage students using CRUD operations.

## Tech Stack

* React
* TypeScript
* Vite
* Git

## Features (Planned)

* Display students in a table
* Add new student
* Edit student
* Delete student with confirmation
* Form validation
* Loading state
* Export table data to Excel

## Project Setup

Clone the repository

```
git clone <https://github.com/Jai-rokkala/Student-crud-app.git>
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open in browser

```
http://localhost:5173
```
